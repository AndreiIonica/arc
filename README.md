## ⚡ Quick start ⚡

```sh
go install github.com/andrei-io//scaffold@latest
```

## Installation

```sh
go get .
```

```sh
make install (only linux works for now)
```

##

## Usage

```sh
scaffold
```

## Run tests

```sh
go test
```
